# Headless IDE (HIDE)

HIDE is an IDE for CodeGen agents.

## Features
- [x] dev containers
- [x] file io
- [ ] lsp
- [ ] search
- [ ] git
- [x] [python client](https://github.com/artmoskvin/hide-python)
- [x] [langchain toolkit](https://github.com/artmoskvin/hide-python)
- [ ] autogen toolkit
- [ ] js/ts client

## Getting started

```shell
make run
```

## Contributing
Run tests
```shell
make test
```
